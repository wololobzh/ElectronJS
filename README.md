# ElectionJS

Jour 1 – Fondamentaux & Architecture
• Introduction à Electron.js : principes, cas d’usage • Initialisation du projet : main.js, preload.js, renderer • Structure d’un projet Electron modulaire • TP : création d’un gestionnaire de notes local

Jour 2 – UI & Communication
• Création d’UI avec HTML/CSS/JS ou React • Responsive design desktop • Communication entre processus (IPC) • TP : ajout d’un système de sauvegarde et de notifications

Jour 3 – Fonctionnalités natives & Sécurité
• Accès au système de fichiers • Stockage local (SQLite, JSON, IndexedDB) • Sécurité : contextBridge, sandboxing, bonnes pratiques • TP : ajout d’un module de configuration utilisateur sécurisé

Jour 4 – Packaging & Déploiement
• Tests et debugging • Packaging avec Electron Forge / Builder • Distribution multiplateforme (Windows, macOS, Linux) • Présentation des projets • TP final : création d’un installateur + documentation technique
